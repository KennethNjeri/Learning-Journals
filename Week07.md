## Content Insights
PHP is a server-side programming language that processes code on the server and sends the result to the browser, unlike HTML which is a markup language handled by the browser enables dynamic and interactive website features like pulling data from databases, running calculations, and managing secure areas.  
PHP code is embedded within opening <?php and closing ?> tags, The echo command in PHP is used to print output to the screen. Variables store information like names, numbers, or other data to be used later in your code, 
starting with a dollar sign followed by the variable name; $x = 5; $y = "John"  
Arrays let you store multiple values in a single variable, making data easier to manage compared to individual variables.  $cars = array("Volvo", "BMW", "Toyota")
Switch statements evaluate a variable against multiple cases and execute the matching case's code.
Loops execute code repeatedly as long as a condition remains true
  •	while - loops through a block of code as long as the specified condition is true
  •	 For loops through a block of code a specified number of times
  •	For each loops through a block of code for each element in an array

A function is a block of statements that can be used repeatedly in a program. A function is not executed automatically when a page loads. A function is executed only when it is called.
